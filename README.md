## プロジェクト概要
microCMSとNext.jsを連携させて簡単なブログを作成しました。

## 使い方
・トップページに今まで書いたブログ記事が表示されているので、タイトルをクリックするとその記事のページに飛びます。<br />
・検索バーでタグの絞り込みを行うことが出来ます。<br />
・ページネーション機能によって過去の記事も見ることが出来ます。<br />
